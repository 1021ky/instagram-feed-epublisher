# 概要

WebアプリでInstagramにSSOログインするとそのユーザーのフィードをepubの形式の電子書籍にしてダウンロードできるものを作りたい。

目的は、Instagramで投稿したコンテンツをまとめて電子書籍にして振り返りやkindle出版へ繋げるようにするため。
自分で研鑽のために100日チャレンジをしたときに、その投稿をまとめて振り返りたいと思ったのがきっかけ。

## 基本機能

ログインは、InstagramのSSOログイン。使う認証は最新のものを使う。

以下の項目でフィードは絞られる。複数指定された場合はAND条件となる。

- ハッシュタグ（指定できるのは1つのみ）
- 期間（開始日と終了日を指定する）
- 最大取得件数（デフォルトは100件、最大500件）
- ユーザー（Instagram APIの制約でログインしているユーザーのみ取得できないため、実質固定）

epubのフォーマットは、book_layoutディレクトリにあるものを参考にする。

## 画面

1画面で完結するシンプルなUIを目指す。

1. ログイン画面
   - InstagramのSSOログインボタン
2. epub作成画面（ログイン後に表示）
   - ハッシュタグ入力欄
   - 期間選択カレンダー
   - 最大取得件数入力欄
   - 「フィードを取得」ボタン
   - ログアウトボタン
   - ファイル保存ダイアログ

## 技術スタック

- フロントエンド: React + TypeScript + Tailwind CSS
- バックエンド: Next.JS + TypeScript
- 認証: BetterAuthのInstagram SSO
- データ取得: Instagram Graph API
- 電子書籍生成: epub.jsまたは類似ライブラリ
- デプロイ: Vercel
